Projetos em Ciência de Dados

Rossmann Sales Prediction

Desafio: O objetivo deste projeto foi desenvolver um modelo preditivo capaz de estimar as vendas das próximas seis semanas para cada loja da rede de farmácias Rossmann. O desafio envolveu lidar com variabilidade nas vendas causada por promoções, feriados, sazonalidade e características regionais das lojas, tornando a previsão manual inconsistente.

Solução: O projeto aplicou técnicas completas de Data Science e Machine Learning, incluindo limpeza e preparação de dados, engenharia e seleção de features, análise exploratória, treinamento e validação de múltiplos modelos de regressão, otimização de hiperparâmetros via busca bayesiana e deploy do modelo em produção com pipeline de dados e API para previsões automatizadas. O XGBoost Regressor se destacou como modelo final, entregando alta precisão e capacidade de generalização.

Tecnologias Utilizadas

  • Python: Pandas, NumPy, Scikit-learn, XGBoost
  • Machine Learning: Regressão Linear, Lasso, Ridge, Random Forest Regressor, XGBoost Regressor
  • Análise de Dados: Feature Engineering, Feature Selection, Exploratory Data Analysis (EDA)
  • Model Deployment: Data Pipeline, Flask API
  • Versionamento e Colaboração: Git, GitHub

Ensaio de Machine Learning

Desafio:Nesse projeto, foram explorados os principais conceitos do treinamento, ajuste de parâmetros e limiares entre overfitting e underfitting de vários algoritmos de Machine Learning dentro das tarefas de classificação, regressão e agrupamento, através de um ensaio de Machine Learning.

Solução: O resultado final do projeto foi um painel mostrando a performance desses algoritmos de Machine Learning, a partir da variações dos valores dos principais parâmetros de cada algoritmo, que controlam o estado de overfitting e underfitting.

Tecnologias Utilizadas

  • Python: Pandas, Scikit-learn
  • Algoritimos de Classificação: KNN, Decision Tree, Random Forest e Logistic Regression
  • Algoritimos de Regressão: Linear Regression, Decision Tree Regressor, Random Forest Regressor, Polynomial Regressor Ridge, Lasso e Elastic Net
  • Algoritimos de Clustering: Affinity Propagation e K-Means
