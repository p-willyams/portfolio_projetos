Projetos em Ciência de Dados

Programa de Fidelidade com Clustering de Clientes

Desafio: Desenvolver um programa de fidelidade personalizado para maximizar a receita da empresa através da segmentação inteligente de clientes.

Solução: Implementei algoritmos de clustering (K-Means, Hierarchical, DBSCAN) para segmentar clientes baseado em comportamento de compra, criando estratégias personalizadas para cada grupo.

Impacto: Potencial de R$ 15MM em receita anual com a implementação da solução.

Tecnologias Utilizadas

  • Python: Pandas, NumPy, Scikit-learn
  • ML: K-Means, Hierarchical Clustering, DBSCAN
  • Visualização: Matplotlib, Seaborn, Metabase
  • Cloud: AWS (EC2, S3, PostgreSQL)
  • Versionamento: Git, GitHub

Identificação de Imóveis para Investimento

Desafio: Identificar imóveis com potencial de valorização para compra e revenda, maximizando o retorno do investimento.

Solução: Análise exploratória avançada com Python para identificar padrões de preços, localização e características que influenciam a valorização imobiliária.

Resultado: Dashboard interativo com recomendações de investimento baseadas em dados históricos e tendências de mercado.

Tecnologias Utilizadas

  • Python: Pandas, NumPy, Seaborn
  • Visualização: Plotly, Folium (mapas interativos)
  • Deploy: Heroku, Streamlit
  • IDE: Jupyter Notebook, PyCharm
